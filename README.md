# cf-legacy-was

### Build & Run Application

* run Application 
```
$ ./gradlew bootRun
```

* build jar & run Application 
```
$ ./gradlew build
$ java -jar build/libs/todoapi-0.0.1-SNAPSHOT.jar
```

### Usage

* Swagger URL 
```
http://localhost:8080/swagger-ui.html
```
